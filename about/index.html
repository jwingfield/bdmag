About

This site is a proof of concept for an exhibition mini-site created using Jekyll.

It takes data collected from the interpretations in Gallery 4, that were used to create the ‘large print’ book that is available at the front desk. By using various filters this has been used to create a database file that is used to provide the information used to automatically create a web page for each individual artwork on display.

The site can then be made available via the internet, with the aim that anybody with any additional visual needs would be able to visit the site using a browser with a screen reader.

Another possibilty is that temporary exhibitions can be kept online permanently. For example the Don’t Give Up exhibition could have the imagery and text boards published - making the exhibition available to a wider audience and for a longer period of time. The full audio interviews could also be accessable this way.
